# CoreIdentity Platform

Complete AI governance platform for enterprise deployments.

## Components

### Discovery Tools
- Risk assessment questionnaires
- Compliance gap analysis
- Automated proposal generation

### Deployment Automation
- Agent deployment scripts
- Domain configuration tools
- Quick-deploy templates

### Monitoring Dashboard
- Real-time agent activity
- Anomaly detection alerts
- Cost tracking
- Compliance reports

### Customer Onboarding
- Guided setup wizard
- Self-service deployment
- Configuration templates

## Quick Start

### For New Customers
```bash
./products/platform/onboarding.sh
```

### For Cole Hospitality (Pilot)
```bash
./products/deployment/scripts/quick-deploy.sh
```

### Start Monitoring Dashboard
```bash
cd products/dashboard
npx serve src
```
